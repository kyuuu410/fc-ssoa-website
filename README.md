# FC SSOA Website

이 프로젝트는 FC SSOA 웹사이트의 소스 코드를 포함하고 있습니다.

## 배포 주소

- **Live URL**: [https://fc-ssoa-frontend.onrender.com/](https://fc-ssoa-frontend.onrender.com/)
- **Hosting**: Render.com (Free Tier)

## 프로젝트 구조

- `frontend/`: Vite로 빌드된 React 애플리케이션
- `backend/`: Python 백엔드 애플리케이션

## 시작하기

### Frontend

1. `frontend` 디렉토리로 이동:
   ```bash
   cd frontend
   ```
2. 의존성 설치:
   ```bash
   npm install
   ```
3. 개발 서버 실행:
   ```bash
   npm run dev
   ```

### Backend

1. `backend` 디렉토리로 이동:
   ```bash
   cd backend
   ```
2. 의존성 설치:
   ```bash
   pip install -r requirements.txt
   ```
3. 서버 실행:
   ```bash
   python main.py
   ```
